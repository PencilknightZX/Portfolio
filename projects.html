My Projects

Explore some of the exciting projects I’ve worked on, showcasing my technical skills and creativity.

Cloud Access Hub

This application is a FastAPI-based backend system to manage access to cloud services based on user subscriptions. It provides role-based access control (RBAC), usage tracking, and enforcement of API limits.

Technologies Used: FastAPI, Python, MongoDB, Uvicorn, Motor, Pydantic

Challenges Faced: Implementing real-time usage tracking and ensuring seamless integration with MongoDB.

EzPaint

A C++ drawing application built using wxWidgets, supporting image saving and loading. It enables users to draw, save their work as PNG or XML, and reload for further editing.

Technologies Used: C++, wxWidgets

Challenges Faced: Handling XML parsing and ensuring cross-platform compatibility of the application.
